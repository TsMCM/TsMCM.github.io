绘图工具

2023-12-14 3 分钟阅读时长

常见的绘图工具

1. ProcessOn - 在线绘图网站

可用于流程图、思维导图、UML、BPMN、网络拓扑图等多种图形绘制，上手很快，比较方便，不需要下载软件就很舒服。
自前，这个网站提供的免费版本的功能用于建模比赛是足够的，大家可以试试。

2. Microsoft Visio - 流程图制作工具和绘图软件

内置很多专业模版，功能强大，更适合专业人士使用。体验上我个人感觉比ProcessOn要难用一点，另外Visio是Office下的一款产品，但安装时确会容易出问题，请注意！

3. Ximind - 思维导图制作工具和绘图软件

这个软件也是付费的，只能绘制思维导图，用起来非常方便,但市面上同类的替代品也很多，大家如果需要绘制思维导图也可以选择其他的软件。

4. AxGlyph - 矢量绘图软件

可以绘制带有数学公式的图形，做物理类型的题目应该能用上，可以用PPT代替它。这个软件学起来还是有一点难度的，需要购买使用，但不是很贵，试用版本有水印。

5. 亿图图示 - 一站式办公绘图利器

一个很强大的画图软件，内置很多精美的画图模版，使用也很方便。
缺点是比较贵，试用版导出图片有水印，如果没有特殊需求建议用其他的软件去代替。

6. GenGebra

GeoGebra是一款完全免费的类似几何画板的 动态 数学软件，用它来画几何图形非常方便,数学建模上面可能用不到，有需要可以下载。（我们提交的论文不支持动态）

7. Apache ECharts - 基于 JavaScript 的开源可视化图表库

想做数据可视化这一块的同学可以了解下，直接学习Echarts需要一定的前端编程基础。
对这个感兴趣的同学可以去学习Python中的 pyecharts，比直接使用Echarts要方便

8. 商业智能（BusinessIntelligence，简称：Bl）

商业智能（BusinessIntelligence，简称：Bl）又称商业智慧或商务智能，指用现代数据仓库技术、线上分析处理技术、数据挖掘和数据展现技术进行数据分析以实现商业价值，例如双十一淘宝可视化大屏展示销售额就是商务智能的应用场景之一。
自前市面上比较有名的两款商业智能分析软件是Tableau和PowerBl。

1 Tableau - 商业智能和分析软件

Tableau可以帮助人们快速分析、可视化、分享数据。借助其拖放功能，无需深入了解数据结构也可快速将数据可视化。Tableau是商业软件，下载安装后可免费试用14天

2. Microsoft Power BI

PowerBI是微软公司出品的一款软件，其的核心理念就是让用户不需要强大的技术背景只需要掌握Excel这样简单的工具就能快速上手商业数据分析及可视化。PowerBI有免费人人桌面版本，也有为企业提供的付费版本

对于基础的建模绘图而言，免费版本足够大家使用了

9. 社会网络图

(1) 不需要编程，手动创建图形的软件： Gelphi和 yEd-Graph Editor;
(2) 借助开源软件的第三方包，例如：Python的 NetworkX和R语言的 igraph

10. 地理数据可视化

地理信息系统（GeographicInformationSystem，GIS）软件可以用于地理数据的可视化，常见的GIS软件有 ArcGIS、 MapGIS、 SuperMap。但这些软件过于专业，入手起来比较困难。前面介绍的Tableau和PowerBl也支持简单的地图可视化。
Python的 folium也支持该功能，喜欢用Python的同学可以学习下。
